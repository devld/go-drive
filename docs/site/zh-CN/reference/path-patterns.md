# 路径模式

go-drive 的搜索过滤、隐藏属性、任务源路径和事件触发器使用 doublestar 风格路径模式。路径不以 `/` 开头，分隔符使用 `/`。

| 模式 | 含义 |
| --- | --- |
| `*` | 匹配一个目录层级内任意数量的非 `/` 字符 |
| `?` | 匹配一个非 `/` 字符 |
| `**` | 跨越零个或多个目录层级 |
| `[abc]` | 匹配字符集合中的一个字符 |
| `{a,b}` | 匹配备选项（在支持该语法的场景） |

示例：

```text
a/*.js       匹配 a/x.js，不匹配 a/sub/x.js
a/**/*.js    匹配 a/x.js 和 a/sub/x.js
**/.git/**   匹配任意层级的 .git 内容
photos/202?  匹配 photos/2020 到 photos/2029
```

不同功能会在模式外增加自己的语法：搜索过滤每行还需以 `+` 或 `-` 开头；缩略图映射使用 `tag:<pattern>`。

删除和移动任务使用通配符时先在测试目录验证。`**` 可能匹配非常大的目录树。
